2️⃣  DECISIONES DE DISEÑO UX/UI
================================================================================

🎯 ESTRUCTURA DE LA PÁGINA:

1. HERO SECTION (Impacto Inmediato)
   ├─ Headline poderoso: "Tu Cocina, Tu Transformación"
   ├─ Subheadline empático con propuesta clara
   ├─ CTA principal con diseño cautivador
   └─ Ingredientes flotantes como elemento visual emocional
   
   DECISIÓN: La forma de división en 2 columnas (left: contenido, right: visual)
   crea balance visual. Los ingredientes flotantes humanizaban la página y generan
   curiosidad inmediata.

2. HISTORIA / STORYTELLING (Conexión Emocional)
   └─ Narrativa breve que habla del VERDADERO valor de cocinar
   
   DECISIÓN: Usar lenguaje emocional ("acto de amor propio") activa los
   receptores de empatía del visitante. El párrafo destacado refuerza el
   mensaje central y crea un momento de pausa reflexiva.

3. BENEFICIOS (Gatillos Mentales)
   ├─ 4 tarjetas con iconos animados
   ├─ Cada beneficio direccionado a necesidades reales
   └─ Animaciones que atrapan la atención
   
   DECISIÓN: 4 beneficios clave = máxima retención. Icons animados mantienen
   el engagement dinámico. Los hover effects invitan a la interacción.

4. GALERÍA DE RECETAS (Prueba Social Visual)
   ├─ 6 tarjetas con recetas reales
   ├─ Emojis en lugar de fotos (estilo moderno, accesible, único)
   └─ Efectos hover dramáticos (zoom + overlay)
   
   DECISIÓN: Los emojis crean una estética lúdica y moderna. Diferencia de
   otros sitios convencionales. El zoom en hover genera sensación de "vida".

5. MINI JUEGO (Momento de Diversión - CLAVE)
   ├─ Juego interactivo: "Atrapa los Ingredientes"
   ├─ Mecánica simple pero adictiva
   └─ Recompensa emocional al final
   
   DECISIÓN: Juegos generan dopamina. El usuario se quedará en la página más
   tiempo. Crea conexión más profunda que solo lectura.

6. TESTIMONIOS (Prueba Social Real)
   ├─ 3 testimonios diversos (edades y perfiles)
   └─ Avatares animados para humanizar
   
   DECISIÓN: Diversidad de edades (42, 38, 65) demuestra que la propuesta
   es para TODOS. Crean confianza y reducen fricción antes de conversión.

7. CTA FINAL (Urgencia y Recompensa)
   ├─ Mensajísimo poderoso: "La Cocina Te Espera"
   ├─ 2 CTAs para diferentes intenciones
   └─ Fondo con gradient emocional
   
   DECISIÓN: Doble CTA permite captar diferentes tipos de users:
   - CTA 1: Action takers (Ver recetas ahora)
   - CTA 2: Nurture ready (Recibir inspiración semanal)

8. FOOTER (Cierre Confiable)
   └─ Información clara sin abrumar
   
   DECISIÓN: Footer minimalista pero funcional. Refuerza marca al final.

5️⃣  FUNCIONAMIENTO DEL MINI JUEGO
================================================================================

🎮 "ATRAPA LOS INGREDIENTES" - Mecánica Completa:

OBJETIVO:
Usuario controla una canasta (🧺) moviéndola hacia los ingredientes que caen
del cielo. El juego transmite la sensación de "estar listo para cocinar".

ELEMENTOS:

1. ÁREA DE JUEGO (.game-area)
   └─ Contenedor rectangular de 400px altura (responsive a 300px en tablets)
   └─ Fondo con patrón dashed para indicar zona interactiva
   └─ Sombra interna para profundidad

2. INGREDIENTES CAYENDO (.falling-item)
   └─ 10 emojis: 🥕🍅🥬🍌🍓🥕🥒🍊🥦🍓
   └─ Animación: @keyframes fall (3s, cae desde arriba hacia abajo)
   └─ Diferentes delays para efecto cascada natural
   └─ Posicionamiento horizontal distribuido: left: calc(10% * var(--item-num))

3. CANASTA DEL JUGADOR (.basket)
   └─ Emoji 🧺 posicionado en bottom: 20px
   └─ Se controla via JavaScript: mousemove event listener
   └─ Sigue suavemente la posición del cursor del usuario

INTERACTIVIDAD (JavaScript Mínimo):

```javascript
gameArea.addEventListener('mousemove', (e) => {
    const rect = gameArea.getBoundingClientRect(); // Obtiene el área de juego
    const x = e.clientX - rect.left;                // Posición del mouse
    basket.style.left = (x - 25) + 'px';            // Mueve la canasta
    gameMessage.textContent = '⬆️ ¡Atrapa esos ingredientes!';
});

setTimeout(() => {
    document.querySelector('.game-success-message').style.display = 'block';
}, 3000); // Muestra mensaje de éxito después de 3 segundos
```

¿POR QUÉ FUNCIONA?

1. SIMPLICIDAD: No hay "puntos reales" que contar. La diversión es el juego.
2. ENTENDIMIENTO INMEDIATO: Todos entienden "mover canasta, atrapar cosas"
3. GRATIFICACIÓN RÁPIDA: El usuario se siente "maestro" en 3 segundos
4. CARGA COGNITIVA BAJA: No interfiere con el mensaje principal
5. ACCESIBILIDAD: El juego es visual pero tiene fallback de JavaScript

MENSAJE DE ÉXITO:
"✨ ¡Excelente! ¡Eres un maestro culinario! 
Ahora que dominas los ingredientes... 
¿Te atreves a cocinar?"

Este mensaje:
✓ Celebra al usuario (recompensa emocional)
✓ Refuerza que "ya está listo"
✓ Pregunta directa que invita a la siguiente acción

7️⃣  ANIMACIONES IMPLEMENTADAS
================================================================================

⚙️ LISTADO COMPLETO DE @KEYFRAMES:

1. @keyframes fadeInUp
   └─ Entrada suave desde abajo con fade
   └─ Usado en: hero-content, story-section, todos los cards

2. @keyframes float
   └─ Movimiento flotante vertical + rotación
   └─ Usado en: ingredientes flotantes del hero
   └─ Efecto: sensación de levedad y movimiento natural

3. @keyframes pulse
   └─ Variación de opacidad (breatear)
   └─ Usado en: iconos de beneficios al hover
   └─ Efecto: atrae atención, indica interactividad

4. @keyframes glow
   └─ Variación de box-shadow con brillo
   └─ Usado potencialmente: botones en hover
   └─ Efecto: elemento "brilla" cuando se interactúa

5. @keyframes fall
   └─ Caída de top a bottom con fade
   └─ Usado en: ingredientes del juego
   └─ Efecto: simulación realista de gravedad

6. @keyframes bounce
   └─ Movimiento de rebote vertical
   └─ Usado en: logo, iconos animados
   └─ Efecto: vitalidad, energía constante

7. @keyframes shake
   └─ Movimiento horizontal rápido
   └─ Disponible para: elementos que necesiten "llamada de atención"
   └─ Efecto: urgencia, movimiento nervioso

8. @keyframes slideInLeft / slideInRight
   └─ Entrada lateral con fade
   └─ Disponible para: elementos que entran por scroll
   └─ Efecto: dirección, flujo visual

9. @keyframes scaleIn
   └─ Entrada con zoom + fade
   └─ Usado en: cards, elementos importantes
   └─ Efecto: aparición energética

10. @keyframes slideInUp
    └─ Entrada desde abajo (específica para mensaje de juego)
    └─ Usado en: .game-success-message
    └─ Efecto: celebración visual

TRANSICIONES APLICADAS:
✓ --transition-fast: 0.2s (hover inmediatos, cambios rápidos)
✓ --transition-base: 0.3s (transiciones normales, interactividad)
✓ --transition-slow: 0.5s (animaciones de entrada/salida)

ANIMATION DELAYS:
✓ Cada elemento tiene delay escalonado
✓ Ejemplo: Benefits cards con delays 0.3s, 0.4s, 0.5s, 0.6s
✓ Efecto: entrada en cascada, más orgánico y hermoso
